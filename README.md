**BetMaster Pro** is a specialized financial calculator designed for **sports arbitrage and advanced betting strategies**.

It’s less of a "bookmaker" where you place bets and more of a **technical suite** for data-driven decision-making. Here is a breakdown of how it functions:

### Key Functionalities
* **Arbitrage Finder:** The core feature shown is an "Arb Calc," which identifies "sure bets." It calculates how to distribute a total stake across different outcomes to guarantee a profit regardless of the result.
* **Implied Probability Analysis:** It automatically converts decimal odds into probability percentages to show the "overround" or the margin available.
* **Betting Versatility:** The sidebar indicates tools for multiple strategies beyond simple arbitrage, including:
    * **Matched Betting:** For clearing bookmaker bonuses risk-free.
    * **Odds Converter:** For switching between decimal, fractional, and American formats.
    * **System/Combo Calcs:** For complex multi-bet structures.

### Target Audience
The app is built for **"sharp" bettors** or those interested in the mathematical side of gambling. The interface is clean and professional, focusing on "Guaranteed Profit" and "Target Stakes" rather than the flashy graphics or team colors typical of standard betting apps.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
